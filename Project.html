PROJECT 1: AMONG US

  • Among Us Character
  • Pure CSS & HTML A manually coded project where I recreated an Among Us crewmate using only HTML and CSS.
  • No images, no libraries, and no graphic tools were used — every part of the character, including the body, visor, backpack, feet, and shadows, was created using styled elements.
  • Purpose: To practice pure CSS layout techniques, positioning (relative & absolute), layering with z-index, shape creation using border-radius, and applying transforms, scaling, and opacity effects — all without relying on external tools or assets.
  • What I Learned: Manual CSS shape-building using borders and border-radius Managing complex layouts with relative and absolute positioning Using opacity, z-index, and transform-origin for visual effects Structuring layered elements purely through HTML & CSS
  • Tech Used:
  • ✅ HTML
  • ✅ CSS (pure, no frameworks)
